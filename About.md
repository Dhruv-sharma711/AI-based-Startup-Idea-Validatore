# 🚀 AI-Based Startup Idea Validator

> *Type your startup idea. Get a full business reality check — instantly.*

---

## 📌 What It Does

The **AI-Based Startup Idea Validator** is a full-stack web application that helps aspiring entrepreneurs validate their startup ideas using the power of AI.

Simply type your idea — the system does the rest:

| Feature | Description |
|---|---|
| 🔍 **SWOT Analysis** | Strengths, Weaknesses, Opportunities & Threats — auto-generated |
| 📊 **Market Size Estimate** | TAM/SAM sizing based on your idea's domain |
| 🏆 **Top 3 Competitors** | Real competitor identification with brief profiles |
| ⭐ **Viability Score** | Rated on a scale of **1–10** with reasoning |
| 📄 **Business Brief PDF** | One-page downloadable brief, ready to share |
| 💾 **Idea History** | Save and compare multiple ideas over time |

---

## 🛠️ Tech Stack

```
Frontend   →   React.js + Tailwind CSS
Backend    →   Node.js + Express.js
Database   →   MongoDB (Mongoose)
AI Engine  →   Gemini API
PDF Export →   jsPDF / Puppeteer
```

> **MERN Stack** — MongoDB · Express · React · Node

## ⚙️ Getting Started

```bash
# Clone the repository
git clone https://github.com/your-repo/ai-startup-validator.git

# Install dependencies
cd server && npm install
cd ../client && npm install

# Add environment variables
cp .env.example .env
# → Add your GEMINI_API_KEY and MONGODB_URI

# Run the app
npm run dev   # starts both client and server
```
